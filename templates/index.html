üîó API Endpoints

GET /api/v1/health

Check API and model health status, performance metrics, and system diagnostics.

POST /api/v1/predict

Predict price for a single diamond. Returns price estimate with confidence interval.

POST /api/v1/predict/batch

Predict prices for multiple diamonds in a single request (up to 100 diamonds).

GET /api/v1/info

Get comprehensive API documentation, valid values, and usage examples.

GET /api/v1/model/info

Get detailed model information, feature importance, and training parameters.

POST /api/v1/validate

Validate diamond data without making a prediction. Useful for form validation.
